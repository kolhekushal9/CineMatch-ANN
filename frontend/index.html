🧠 Artificial Neural Network

This system uses Neural Collaborative Filtering (Deep Learning) instead of standard math. It assigns a 50-dimensional Vector (Embedding) to every User and Movie to extract deeply hidden personal movie preferences!


  • 1. Embeddings: We fetch $User \rightarrow Vector_{u}$ and $Movie \rightarrow Vector_{m}$
  • 2. Concatenation: $X = [Vector_{u}, Vector_{m}]$ representing the combined state.
  • 3. Dense Hidden Layers: 3 layers (128, 64, 32 neurons) extract highly complex non-linear habits!
  • 4. Activation: We use ReLU $f(x) = max(0, x)$ which solves the vanishing gradient problem.
  • 5. Final Output: A single continuous predicted rating scalar.
